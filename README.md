# 👟 Outlet Senai — Loja de Tênis

Projeto de e-commerce de tênis desenvolvido como trabalho acadêmico no SENAI.

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica das páginas
- **CSS3** — Estilização com variáveis CSS, gradientes e animações
- **JavaScript** — Lógica do carrinho de compras
- **Google Fonts** — Tipografia (Poppins)
- **Font Awesome** — Ícones

## 📁 Estrutura do Projeto

```
Site-Tenis-Senai-Outlet/
├── index.html                    ← Página principal
├── assets/
│   ├── css/
│   │   ├── style.css             ← Estilos globais
│   │   └── produto.css           ← Estilos da página de produto
│   ├── js/
│   │   └── carrinho.js           ← Lógica do carrinho
│   └── img/
│       ├── produtos/             ← Fotos dos tênis
│       ├── banner.png
│       ├── logo.png
│       └── favicon.png
├── pages/
│   ├── nike-air.html             ← Detalhe: Nike Air
│   ├── adidas-ultra.html         ← Detalhe: Adidas Ultra
│   └── puma-sport.html           ← Detalhe: Puma Sport
├── .gitignore
└── README.md
```

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/Site-Tenis-Senai-Outlet.git
   ```
2. Abra o arquivo `index.html` no navegador

> **Nota:** Por ser um site estático, não precisa de servidor. Basta abrir o HTML diretamente.
